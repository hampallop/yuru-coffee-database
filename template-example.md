# Name:

A way to identify coffee. A combination of;
```
[origin] [owner/farm/region] [varietal] [process]
```

e.g.;
Ethiopia Gelana - Gori Gesha | Natural

# Basic info:

Origin：Ethiopia
Region：Yirgacheffe
Estate: Gelana
Altitude: 1,850-2,000 masl.
Varietal: Gori Gesha
Process: Natural
Tasting Notes：Strawberry, Ripe fruit, Peach, Apple, Grape
